    • Некоторые кнопки имеют шорткаты для клавиатуры
    • ПКМ по изображению в галерее сохранит его
  • Если изображение сохранялось с сайта, то оно имеет определенный формат с двумя ID, например 
    feedback-26399253-32396290-1.jpg
    • Первое число (26399253) означает ID пользователя, которое можно подставить в ссылку. Например: /wb/person/26399253
    • Второе число (32396290) означает ID товара, которое можно подставить в ссылку. Например: /wb/product/32396290
